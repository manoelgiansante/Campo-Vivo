# Assets Placeholder

Este diretório contém os assets do aplicativo Campo Vivo.

## Arquivos necessários:

Para o aplicativo funcionar corretamente, você precisa criar os seguintes arquivos de imagem:

1. **icon.png** (1024x1024px) - Ícone do aplicativo
2. **splash.png** (1284x2778px) - Tela de splash
3. **adaptive-icon.png** (1024x1024px) - Ícone adaptativo para Android
4. **favicon.png** (48x48px) - Favicon para web

### Dica:
Você pode usar ferramentas como Figma, Canva ou geradores online para criar esses assets.

O tema de cores do app é:
- Verde primário: #16a34a
- Verde claro: #22c55e
- Verde escuro: #15803d

Para gerar rapidamente, você pode usar:
- https://www.appicon.co/ para gerar ícones
- https://apetools.webprofusion.com/ para splash screens
